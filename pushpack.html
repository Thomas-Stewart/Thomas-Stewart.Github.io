Push Pack

  • This game started as a fun idea and turned into a great addition to the iOS App Store.

    Created in Unity, this game leverages a robust procedural level generation system where the roads are dynamically created and deformed with splines while obstacles and powerups of varying degrees will spawn according to level difficulty on particular intervals on the mesh.

    The main takeaways I obtained from this project are scriptable mesh deformation, spline editing, raycasting, custom dynamic UI elements, physics materials, and collision optimization.
